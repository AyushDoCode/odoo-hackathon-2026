# AssetFlow by odoo — Login Page

A small, responsive login page for **AssetFlow by odoo**. Designed to be easy to read and customize.

## Tech Stack

- HTML5
- CSS3 (variables)
- Vanilla JavaScript (small DOM helper for password toggle)

## Features

- Clean, responsive login form layout.
- Email + password login with show/hide password toggle.
- All colors exposed as CSS variables in `assets/style.css` for easy theming.
- Minimal, vanilla HTML/CSS/JS — no build tools required.

## Files

- `index.html` — main markup for the login page.
- `assets/style.css` — styles and CSS variables.
- `assets/images/` — SVG assets used by the page.
